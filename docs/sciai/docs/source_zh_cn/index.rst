MindSpore SciAI介绍
===================

MindSpore SciAI是基于昇思MindSpore打造的AI4SCI高频基础模型库，内置了60+高频模型，覆盖度全球第一。

提供了高阶API，开发者和用户开箱即用；精度和性能达到了业界SOTA。

高频模型涵盖物理感知（如PINNs、DeepRitz以及PFNN）和神经算子（如FNO、DeepONet）等主流模型。

应用领域覆盖到了流体、电磁、声、热、固体、生物等众多学科。SciAI为广大开发者和用户提供了高效、易用的AI4SCI计算平台。
