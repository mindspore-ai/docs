## MSFormat

```C
typedef enum MSFormat {
  kMSFormatNCHW = 0,
  kMSFormatNHWC = 1,
  kMSFormatNHWC4 = 2,
  kMSFormatHWKC = 3,
  kMSFormatHWCK = 4,
  kMSFormatKCHW = 5,
  kMSFormatCKHW = 6,
  kMSFormatKHWC = 7,
  kMSFormatCHWK = 8,
  kMSFormatHW = 9,
  kMSFormatHW4 = 10,
  kMSFormatNC = 11,
  kMSFormatNC4 = 12,
  kMSFormatNC4HW4 = 13,
  kMSFormatNCDHW = 15,
  kMSFormatNWC = 16,
  kMSFormatNCW = 17
} MSFormat;
```

| -------- | --- | ------ |
| kMSFormatNCHW     | 0   | NCHW   |

| kMSFormatHWKC     | 3   | HWKC   |
| kMSFormatHWCK     | 4   | HWCK   |
| kMSFormatKCHW     | 5   | KCHW   |
| kMSFormatCKHW     | 6   | CKHW   |
| kMSFormatKHWC     | 7   | KHWC   |
| kMSFormatCHWK     | 8   | CHWK   |
| kMSFormatHW       | 9   | HW     |
| kMSFormatHW4      | 10  | HW4    |
| kMSFormatNC       | 11  | NC     |
| kMSFormatNC4      | 12  | NC4    |
| kMSFormatNC4HW4   | 13  | NC4HW4 |
| kMSFormatNCDHW    | 15  | NCDHW  |
| kMSFormatNWC      | 16  | NWC    |
| kMSFormatNCW      | 17  | NCW    |
