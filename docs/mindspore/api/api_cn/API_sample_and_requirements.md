## 注意事项

1. 链接的用法：

    如果链接文本是网址，语法如下：

    ```text
    `tensor <https://www.gitee.com/mindspore/mindspore/blob/r1.5/mindspore/common/tensor.py>`_
    ```

    请注意，链接文本和 URL 的开头 < 之间必须有一个空格。

2. 表格的用法：

    表格必须包含多行，并且第一列单元格不能包含多行。如下所示：

    ```text
        =====  =====  =======
         A           B            A and B
        =====  =====  =======
        False       False      False
        True        False      False
        False       True       False
        True        True       True
        =====  =====  =======
    ```

3. 注意文中专有名词的正确书写，例如“Numpy”，“Python”，“MindSpore”等。
