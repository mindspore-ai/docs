# 安装MindSpore Vision

<a href="https://gitee.com/mindspore/docs/blob/r1.7/docs/vision/source_zh_cn/mindvision_install.md" target="_blank"><img src="https://mindspore-website.obs.cn-north-4.myhuaweicloud.com/website-images/r1.7/resource/_static/logo_source.png"></a>

## 环境依赖

- numpy 1.17+
- opencv-python 4.1+
- pytest 4.3+
- [MindSpore](https://www.mindspore.cn/install) >=1.7.0 <=1.8.1
- ml_collection
- tqdm
- pillow

## 安装

### 环境准备

- 创建一个conda虚拟环境并且激活。

    ```shell
    conda create -n mindvision python=3.7.5 -y
    conda activate mindvision
    ```

- 安装MindSpore

    ```shell
    pip install mindspore
    ```

### 安装MindSpore Vision

- 使用pip安装

    ```shell
    pip install mindvision
    ```
