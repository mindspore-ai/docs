MindSpore Recommender Documents
================================

MindSpore Recommender是一个构建在MindSpore框架基础上，面向推荐领域的开源训练加速库，通过MindSpore大规模的异构计算加速能力，MindSpore Recommender支持在线以及离线场景大规模动态特征的高效训练。
